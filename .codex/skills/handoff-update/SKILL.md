---
name: handoff-update
description: Update execution state, progress, decisions, failures, and next-step notes so long-running work can resume cleanly.
---

# Goal
Leave the repository in a resumable state after each meaningful step.

# Required Updates
- STATE.md
- TASKS.md
- HANDOFF.md
- DECISIONS.md when a real choice was made
- FAILURES.md when something failed

# Output
Summarize:
- current state
- completed work
- blocker if any
- next recommended action
